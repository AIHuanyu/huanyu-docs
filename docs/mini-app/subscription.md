## 1. 关于公众号
公众号是为媒体和个人提供一种新的信息传播方式，主要功能是在微信侧给用户传达资讯；（功能类似报纸杂志，提供新闻信息或娱乐趣事）  
适用人群：个人、媒体、企业、政府或其他组织。  
群发次数：公众号（认证用户、非认证用户）1天内可群发1次消息。  

- [公众号开发文档](https://developers.weixin.qq.com/doc/subscription/guide/)
- [公众号问题答疑](https://kf.qq.com/faq/120911VrYVrA15091832Qzqq.html)

## 2. 注册申请
在微信公众平台点击【立即注册】申请公众号；
### 2.1 基本信息
基本信息需要填写邮箱和密码，密码需要符合微信公众平台的密码要求；
::: tip
一个邮箱只能用于注册一个公众号
:::
### 2.2 选择类型
该步可以选择公众号、服务号、企业微信三种类型，一旦选择创建成功后则无法修改。

### 2.3 信息登记
这一步需要先选择微信公众账号主体类型，包括政府、企业、其他组织或个人。若为个人运营的公众号，请选择“个人”类型。

选择“个人”主体后，需完成以下步骤：
- 填写主体信息：包括身份证上的姓名和身份证号码；
- 绑定管理员微信：使用管理员本人的微信扫码，将微信账号与公众号进行绑定；
- 登记联系方式并验证：填写管理员的手机号码，并通过短信验证码完成验证。

::: tip
主体类型：个人可注册1个账号，个体工商户、企业、其他组织可注册2个账号，政府和媒体可注册50个账号；  
手机号码：一个手机号码只能注册5个公众账号；
:::
### 2.4 公众号信息
这一步需要填写以下信息：
- 账号名称：4~30个字符（1个汉字算2个字符）；
- 功能介绍：4~120个字，介绍此公众账号功能与特色；
- 内容类目：选择此公众账号的服务类目，至少选择1个内容类目，最多可选择3个内容类目；
- 运营地区：选择此公众账号的运营地区，精确到区县级；

点击完成即可完成公众号的注册。

## 3. 基本信息设置
注册成功后，还需要设置以下信息：
- 账号头像
- 微信号（如果不设置微信号，公众号会默认展示公众号ID）
::: info
设置成功后，一年内可再申请修改1次微信号  
设置成功后，微信用户可通过该微信号搜索到你的公众账号  
可以使用6-20个字母、数字、下划线和减号，必须以字母开头
:::
## 4. 账号注销
- 账号注销流程中会有7天的冻结期，在冻结期可以通过管理员账号登录取消注销；
- 冻结期间账号资源不会释放，因此，无法重新注册新的个人账号；
