## 1. 邮箱激活
进入 [微信公众平台官网](https://mp.weixin.qq.com/)，点击 “立即注册”，选择 “小程序” 类型。  
::: info
邮箱要求：必须使用一个 从未在微信平台（包括公众号、小程序、视频号等）注册或绑定过的邮箱地址。
:::
## 2. 设置登录密码
通过邮箱验证后，需设置登录微信公众平台的密码：
::: info
- 密码要求最短8位，区分大小写；
- 密码应包含 大小写字母 + 数字 + 英文符号；
:::

## 3. 主体实名认证
注册账号需绑定真实身份信息。对于 个人开发者，请选择 “个人” 主体类型，并按要求填写以下信息：
- 真实姓名（须与身份证一致）
- 身份证号码
- 手机号（用于接收短信验证码）
- 微信扫码确认身份：使用本人实名认证的微信账号扫描二维码，完成最终身份核验。

## 4. 填写基本信息
上述步骤完成后，即表示注册成功，注册成功后还填写小程序基本信息
- 小程序名称：名称需符合命名规范，不能与其他小程序重名；
- 头像：上传清晰、合规的图标；
- 简介：简明扼要描述小程序的功能或服务内容。

::: tip
小程序名称修改后会有2天的冻结期，在冻结期内该主体可以使用原名称注册。
:::
## 5. 设置小程序类目
类目决定了小程序可提供的服务范围和功能权限，后续还可以调整（小游戏除外）；
::: warning
注意事项：非游戏类目一旦设置成功，你将不得修改为游戏类目。
:::
## 6. 小程序备案与微信认证
完成上述步骤后，您的小程序账号已创建成功，但若计划正式上线运营，还需关注以下两个关键环节：
小程序备案和微信认证，后续文章会详细介绍。
