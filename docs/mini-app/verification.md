## 1. 作用
微信认证主要目的在于为账号解锁“被搜索”与“被分享”两大关键能力。未完成认证虽不影响版本正常发布，但账号会失去搜索曝光和社交分享的权限，即便发布也难以触达用户，相当于“上线即隐身”，无法发挥实际价值。

## 2. 认证流程
微信认证流程非常简单，仅需以下几步：
1. 登录微信小程序后台，找到“微信认证”；
2. 同意协议、填写资料、确认名称、填写发票信息；
3. 缴费;
4. 等待审核：微信会安排第三方审核机构对你的认证信息进行审核，审核周期通常为 1-5 个工作日。

::: tip
个人开发者认证费用为 30 元，企业等其他主体类型的认证费用一般为 300 元。
:::

## 3. 审核过程
初次可能会不通过，必须电话沟通确认信息后才通过。

当天或者第二天就可以完成认证。
